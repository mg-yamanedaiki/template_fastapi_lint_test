# appディレクトリ構成

| ディレクトリ名 | 役割                     |
| -------------- | ------------------------ |
| auth           | 認証                     |
| constants      | 定数                     |
| models         | DBテーブル定義           |
| routes         | ルーティング             |
| schemas        | スキーマ(形式)定義       |
| services       | ビジネスロジック,DB連携  |
| utils          | 全体で使用する便利処理等 |
